## 讲师介绍

张旭，矩阵起源资深研发工程师。国内第一个 Go 开源 API 网关 [Manba](https://github.com/fagongzi/manba) 作者，弹性强一致 NoSQL 数据库 [ElastiCell](https://github.com/deepfabric/elasticell) 作者，分布式存储框架 [MatrixCube](https://github.com/matrixorigin/matrixcube) 主要贡献者，对分布式方向有深入研究和实践，目前负责 MatrixOne 分布式架构，包含集群管理、分布式事务、分布式共识日志等的开发工作。

## 演讲主题

在代码检测与稳定性测试中，高效 Debug 数据库事务正确性

## 演讲提纲

- MatrixOne 架构简介
- 事务正确性相关 BUG 与当前应对方案
- 救赎之旅——如何高效 DEBUG 数据库事务正确性

- 日期：2024 年 3 月 16 日
- 时间：16:40-17:15
- 地点：北京中关村创业大街南口拓荒族·三层会议室

### 关于 OSChina 源创会

OSC 源创会是开源中国社区（oschina.net）主办，聚焦开源、创新的技术沙龙。源创会始终秉承“自由、开放、分享”的宗旨，聚集最优质的技术资源与行业案例，对话最优秀的技术领军人物，为广大开发者带来最新开源技术、前沿技术视角、以及落地实践经验。
