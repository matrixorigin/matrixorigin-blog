## NVIDIA Launches Quantum Cloud Service

## Google Merges Angular and Wiz Frameworks

The Angular team recently officially announced the merger of Angular and Wiz to meet developers' hybrid needs. Wiz is Google's internal JavaScript framework, created by Malte Ubl more than a decade ago, and currently supports applications such as Search, Gmail, and Photos. Angular and Wiz serve different application domains. Wiz focuses on performance-critical applications such as Google Search, while Angular targets highly interactive applications and prioritizes developer experience and rapid delivery of complex UIs. Examples of Angular usage include Gemini and Google Analytics.

## Recent Java News: New OpenJDK

Oracle Java architect Paul Sandoz submitted JEP Draft 8326878, Vector API (Incubator). After seven incubation rounds from JDK 16 to JDK 22, this JEP proposes to re-incubate the API in JDK 23, with no API changes or substantial implementation changes compared with JDK 22. This feature will introduce an API for expressing vector computations that reliably compile at runtime to optimal vector instructions on supported CPU architectures, thus achieving performance superior to equivalent scalar computations.

## DeepMind Co-Founder Suleyman Becomes CEO of Microsoft's New AI Division

On Tuesday local time in the United States, Microsoft announced that it had hired Mustafa Suleyman, co-founder of AI startup DeepMind, to lead its newly formed artificial intelligence division, responsible for projects such as Copilot. Suleyman will oversee a series of key projects, including integrating the AI assistant Copilot into Windows and introducing conversational elements into the Bing search engine. This move marks Microsoft's active effort to defend against challenges from Google in the competitive AI product market.

## OpenAI Removes Registration Requirement for ChatGPT

OpenAI announced that it will allow users to use ChatGPT directly without registering for the service, making it easier for people to experience the potential of artificial intelligence. OpenAI said it will gradually roll out this feature starting Monday, allowing more than 100 million users in 185 countries and regions to use ChatGPT to learn new things, find creative inspiration, and get answers to questions.
