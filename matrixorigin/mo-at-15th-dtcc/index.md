# Introduction

From August 22-24, 2024, the 15th China Database Technology Conference (DTCC 2024), organized by IT168 in conjunction with the IT community sites ITPUB and ChinaUnix, was grandly held at the Beijing Langyue Sishan Garden Hotel. The conference, themed “Indigenous Innovation, Intelligent Digital Future,” kept pace with the times and led the way in cutting-edge technology. It brought together the best practices of database applications across various industries, showcasing how technology helps businesses overcome challenges, enhance efficiency, and drive innovation.

Matrix Origin was invited to participate in this event, where it showcased the hyper-converged heterogeneous cloud-native database MatrixOne. The company engaged with the audience in discussions on how AI-driven databases are building the next generation of intelligent data management infrastructure. One of the highlights was MatrixOne's integration with large language models, specifically the Matrix Search enterprise-level multimodal data search platform, which drew significant interest from attendees. The booth attracted an unprecedented number of visitors, intrigued by the application of hyper-converged vector databases combined with large models.

# Hyper-Converged Heterogeneous Cloud-Native Database MatrixOne

MatrixOne is a multi-model database based on cloud-native technology that can be deployed in both public and private clouds. Through an innovative architecture designed and developed with separation of storage and computing, separation of read and write operations, and separation of hot and cold data, users can support various workloads such as transactions, analytics, streaming, time-series, and vectors on a single storage and computing engine. This architecture allows real-time, on-demand sharing or isolation of storage and computing resources.

Leveraging the innovative HSTAP (Hybrid Streaming, Transactional, and Analytical Processing) architecture, MatrixOne aims to reduce data fragmentation, creating a simplified “One Size Fits Most” data management and application development solution. This helps users significantly simplify increasingly complex architectures, providing simple, flexible, fast, and cost-effective data services.
