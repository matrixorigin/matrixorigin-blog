## **Introduction to MatrixOne Intelligence**

MatrixOne Intelligence is an AI-powered data intelligence platform for multimodal data, designed to help enterprises tackle challenges such as data fragmentation, complex integration of multimodal data, and difficulties in implementing GenAI applications. Through data ingestion, intelligent parsing, data workflows, and a hyper-converged lakehouse foundation, MatrixOne Intelligence provides enterprises with a one-stop, end-to-end platform. It transforms internal proprietary data into AI-ready data to serve GenAI applications. Based on an innovative cloud-native architecture and storage-compute separation design, the platform supports unified management and efficient processing of both structured and unstructured data. It offers highly flexible deployment capabilities, adaptable to public clouds, private clouds, and on-premises data centers.  
MatrixOne Intelligence is committed to helping enterprises fully unlock the potential of private domain data, ensuring this data becomes a key source of unique competitive advantage in the AI era.

## **Version 3.2 Major Upgrade: Smarter, More Collaborative, More Powerful**

### I. Comprehensive Workflow Upgrade: Building High-Quality AI Data Pipelines

A full upgrade of data workflow capabilities—from data parsing and cleansing to augmentation—establishes end-to-end high-quality AI data pipelines, empowering enterprises to efficiently build AI training corpora and knowledge assets.  
● **Audio/Video Parsing Operators**: Support automatic parsing of more common media formats, enabling enterprises to convert customer service recordings, meeting videos, educational materials, etc., into high-value structured content. This bridges the critical path from unstructured data to knowledge bases.  
● **Data Cleansing Operators**: Built-in capabilities including deduplication, desensitization, and format standardization ensure data accuracy and usability.  
● **Data Augmentation Operators**: One-click generation of Alpaca-format datasets suitable for instruction fine-tuning, dialogue training, and other scenarios, accelerating model optimization!
![](/content/en/moi-3.2-release/1.png)

### II. Multi-User Collaboration: Flexible Control, Efficient Governance

Introduces a flexible permission management mechanism through the RBAC model, enabling fine-grained permission control for multi-user collaboration to meet data governance needs across organizations of all sizes.  
● **RBAC (Role-Based Access Control) Model**: A three-tier "User-Role-Permission" structure enables unified permission assignment and centralized management, enhancing standardization and maintainability.  
● **Centralized Configuration & Permission Reuse**: Users inherit permissions via roles (not direct binding), supporting abstract permission configuration and cross-team reuse.  
● **Adaptability to Diverse Organizations**: Enables efficient, secure data collaboration and governance for both SMEs and large enterprises, improving operational efficiency and compliance.
![](/content/en/moi-3.2-release/2.png)

### III. GenAI Workspace API Key Management: Secure Integration, Hassle-Free Invocation

New API key management feature launched! Enables secure, efficient API calls by third-party apps, automation scripts, and internal services while ensuring controllable permissions and preventable risks.  
● **One-click generation & masked display** prevents key leakage!  
● **Supports key refresh** with unified control over API call permissions for better risk prevention!  
● **Seamless integration** with third-party apps, automation scripts, and internal services!
![](/content/en/moi-3.2-release/3.png)

### IV. New File Export Support: Complete Content, Efficient Archiving

DOC/DOCX, PPT/PPTX, and TXT files can now be exported as **complete, unsegmented Markdown (`full.md`)**, facilitating holistic viewing and management!
