## ▶ DB-Engines 2024 年 3 月数据库排行榜出炉。

本月的 DB-Engines Ranking 共有 418 个产品上线，比上个月增加一个产品。本月 DB-Engines 再一次普跌为主，大部分数据库产品出现了分数普遍下跌的情况。本月 Oracle 再一次以 -20.39 的成绩领跌榜。年度数据库 PostgreSQL 以 +5.50 的成绩领涨全榜。

_来源：https://www.modb.pro/db/1765256633084645376_

## ▶ 微信即将推出原生 Linux 版本。

根据爆料，微信将迎来 Linux 原生版重构，全新 1.0 版本已开启内测，并放出了测试截图和安装包。微信 Linux 原生版将支持 X86、ARM、龙芯 LoongArch64 架构，系统方面支持麒麟和统信 UO。

_来源：https://www.oschina.net/news/281936/wechat-linux_

## ▶ 亚马逊取消客户迁移云服务的费用。

当地时间 3 月 5 日，亚马逊网络服务公司副总裁罗伯特·肯尼迪在一篇博客文章中表示，亚马逊网络服务公司将不再向希望，从该公司服务器中提取所有数据并将其转移到其它云服务提供商的客户收取费用。

_来源：https://www.niutoushe.com/lives/yamaxunquxiaokehuqianyiyunfuwudefeiyong_

## ▶ SQLAlchemy 2.0.28 发布，Python ORM 框架。

SQLAlchemy 是一个 Python 的 SQL 工具包以及数据库对象映射 (ORM) 框架。它包含整套企业级持久化模式，专门用于高效和高性能的数据库访问。

_来源：https://www.oschina.net/news/281564/sqlalchemy-2-0-28-released_

## ▶ Oracle Globally Distributed Autonomous Database 正式推出。

Oracle 宣布正式推出 Oracle Globally Distributed Autonomous Database。Globally Distributed Autonomous Database 基于 Oracle 备受肯定的分片技术构建而成，不仅为客户提供 Oracle Autonomous Database 的优势，同时也让客户能够控制数据分发和数据放置政策。

_来源：https://www.modb.pro/db/1765197003038986240_

## ▶ 谷歌设立500万美元奖金，旨在寻找量子计算机的实际用途。

谷歌和 XPRIZE 基金会正在发起一项价值 500 万美元（约 3605 万元人民币）的竞赛，旨在寻找量子计算机的实际应用，从而“真正造福社会”。据了解，这项为期三年的竞赛于美国东部时间 3 月 4 日启动，目标是将量子计算的奥妙转化为解决现实世界问题的方案。该竞赛具有广泛的比赛范围和明确的目标，参赛者的任务包括开发新的量子算法、将现有算法应用于未知领域。

_来源：https://www.niutoushe.com/lives/gugesheli500wanmeiyuanjiangjinzhizaixunzhaoliangzijisuanjideshijiyongtu_

## ▶ 英特尔宣布成立全新独立运营的 FPGA 公司——Altera。

3 月 1 日，英特尔宣布成立全新独立运营的 FPGA 公司——Altera。Altera 首席执行官 Sandra Rivera 介绍，Altera 的解决方案专为包括网络、通信基础设施、低功耗嵌入式等在内的广泛市场和实践应用而优化。

_来源：https://www.niutoushe.com/lives/yingteerxuanbuchengliquanxinduliyunyingdefpgagongsialtera_

## ▶ OpenAI 最强竞争对手 Anthropic 发布 Claude 3。

3 月 5 日，大模型公司 Anthropic 发布了其第三代人工智能（AI）模型——Claude 3 系列模型，包括 Claude 3 Opus、Claude 3 Sonnet 和 Claude 3 Haiku。其中，Claude 3 Opu 为 Claude 3 系列模型的版本，具有接近人类的理解能力，可以处理开放式提示和复杂的任务。
