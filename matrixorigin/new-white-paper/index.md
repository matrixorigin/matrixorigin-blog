As one of the core foundational software categories comparable to operating systems, databases have continued to evolve with the arrival of the cloud era, moving into the construction and upgrading of cloud-native databases. Since 2023, the continuous evolution of AIGC and **multimodal large model** technologies has further driven deep integration between BI and AI, bringing a new source of innovation to databases.

MatrixOne is a multimodal hyper-converged database based on cloud-native technologies that can be deployed in both public and private clouds. Through its newly designed and developed original architecture featuring storage-compute separation, read-write separation, and hot-cold separation, users can support multiple workloads such as transactions, analytics, streams, time-series, and vectors on one storage and compute engine, while sharing or isolating storage and compute resources in real time and on demand.

MatrixOne Cloud is a fully managed cloud-native data platform built on the MatrixOne kernel. It not only provides zero-threshold capability to create and use MatrixOne product services, but also introduces a new Serverless instance management model and SQL-usage-based billing model, enabling automatic elastic scaling of compute resources. Users no longer need to care about machine configuration and can focus entirely on meeting SQL needs.

This white paper provides a detailed introduction to the technical architecture, core advantages, key capabilities, data ecosystem, O&M solutions, and multiple application-scenario solutions of MatrixOne and MatrixOne Cloud. It aims to help enterprises simplify IT architecture and realize a new paradigm for efficient data management and application development. **In addition, this white paper provides, for the first time, a detailed introduction to how MatrixOne helps enterprise customers solve challenges across different scenarios and industries**, including:

- SaaS application solution: Supports efficient operation and management of enterprise-grade SaaS applications.
- Enterprise application HTAP solution: Supports both OLTP and OLAP to meet enterprise hybrid transactional and analytical processing needs.
- IoT data platform solution: Supports collection, storage, and real-time analytics of massive IoT device data.
- Database containerization solution: Enables containerized database deployment and improves resource utilization efficiency and flexibility.
- Real-time data analytics solution: Provides real-time data processing and analytics capabilities to support enterprise real-time decision-making.
- Large model and AI solution: Supports large model training and inference, helping enterprises implement AI applications.
