## ▶ Oracle 助力企业实现智能化运营。

Oracle 宣布推出 Oracle Smart Operations，这是 Oracle Fusion Cloud Supply Chain & Manufacturing（SCM）中新增的供应链执行功能。在人工智能 （AI）等前沿技术的支持下，Oracle Fusion Cloud Manufacturing 和 Oracle Fusion Cloud Maintenance 中的新功能将能够帮助客户提高生产率、改善质量、减少意外停机时间并提高运营可见性，从而增加工厂产量。

## ▶ 英伟达推出 Quantum Cloud 云服务。

英伟达发布推出 Quantum Cloud 云服务，基于该公司的开源 CUDA-Q™ 量子计算平台，供研究人员在化学、生物学和材料科学等领域测试量子计算软件。Quantum Cloud 云服务首次为用户在云中构建和测试新的量子算法和应用，包括强大的模拟器和量子-经典混合编程工具。

_来源：https://www.niutoushe.com/lives/yingweidatuichuquantumcloudyunfuwu_

## ▶ 谷歌合并 Angular 和 Wiz 框架。

Angular 团队日前正式发文官宣将 Angular 和 Wiz 进行合并，旨在满足开发人员的混合需求。Wiz 是 Google 的内部 JavaScript 框架，由 Malte Ubl 在十多年前创建，目前为搜索、GMail、照片等应用提供支持。Angular 和 Wiz 彼此服务于不同的应用领域。Wiz 专注于性能关键的应用程序，例如 Google 搜索，而 Angular 则面向高度交互的应用程序，优先考虑开发人员体验和复杂 UI 的快速交付。Angular 的使用示例包括 Gemini 和 Google Analytics。

_来源：https://www.oschina.net/news/285844/angular-and-wiz-are-better-together_

## ▶ Java 近期新闻：新的 OpenJDK。

Oracle 的 Java 架构师 Paul Sandoz 已提交 JEP Draft 8326878（Vector API（孵化器））。经过从 JDK 16 到 JDK 22 的七轮孵化后，该 JEP 提议在 JDK 23 中 重新孵化 该 API，相对于 JDK 22 没有 API 方面的变更和实现上的实质性更改。该特新将引入一个 API，用于“表达在受支持的 CPU 架构上可靠地编译为最佳矢量指令的矢量计算，从而实现比等效标量计算更优越的性能”。

_来源：https://www.infoq.cn/article/LxAxaxe3wZ5sM7IYhKod_

## ▶ DeepMind 联合创始人苏莱曼担任微软新 AI 部门 CEO。

美国当地时间周二，微软宣布已聘请人工智能初创公司 DeepMind 的联合创始人穆斯塔法·苏莱曼（Mustafa Suleyman）领导其新成立的人工智能部门，负责 Copilot 等项目。苏莱曼将负责一系列关键项目，包括将人工智能助手 Copilot 集成到 Windows 系统中，以及在必应搜索引擎中引入对话元素。这一举措标志着微软在竞争激烈的人工智能产品市场上，正积极抵御来自谷歌的挑战。

_来源：https://www.niutoushe.com/lives/deepmindlianhechuangshirensulaimandanrenweiruanxinaibumenceo_

## ▶ OpenAI 放开限制，用户无需注册即可使用 ChatGPT。

OpenAI 宣布，将允许用户直接使用 ChatGPT，而无需注册该项服务，这将让人们更加容易体验人工智能的潜力。OpenAI 表示，它将从周一开始逐步推出这一功能，让 185 个国家和地区的超 1 亿用户使用 ChatGPT 来学习新事物、寻找创意灵感，并获得问题的答案。

_来源：https://www.oschina.net/news/285832/chatgpt-no-longer-requires-an-account_
